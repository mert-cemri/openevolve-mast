# Tic Tac Toe Game

Welcome to the Tic Tac Toe Game! This application provides a user-friendly interface for two players to enjoy a classic game of Tic Tac Toe. The game is designed using Python and the Tkinter library, offering a simple yet engaging experience.

## Main Functions

- **Two-Player Mode**: The game allows two players to take turns marking their respective symbols (X or O) on a 3x3 grid.
- **Winner Detection**: The game automatically checks for a winner after each move and announces the winner.
- **Draw Detection**: If all cells are filled and no player has won, the game declares a draw.
- **Reset Functionality**: Players can reset the board at any time to start a new game.

## Installation

### Prerequisites

- **Python**: Ensure that Python is installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Setting Up the Environment

1. **Clone the Repository**: Download the game files to your local machine.
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Install Dependencies**: This project does not require any external dependencies beyond Python's standard library.

## How to Play

1. **Start the Game**: Run the `main.py` file to launch the game.
   ```bash
   python main.py
   ```

2. **Gameplay**:
   - The game window will open with a 3x3 grid.
   - Player X starts the game. Click on any empty cell to place your symbol.
   - Players take turns to place their symbols (X or O) on the grid.
   - The game will automatically check for a winner after each move. If a player wins, a message will be displayed.
   - If all cells are filled and there is no winner, the game will declare a draw.

3. **Reset the Game**: Click the "Reset" button to clear the board and start a new game.

## Troubleshooting
