# Word Grouping Puzzle Game

Welcome to the Word Grouping Puzzle Game! This application challenges users to group sixteen words into four categories of four words each, following the standard logic of the NYT Connections game. The game provides an engaging way to test your categorization skills and logical thinking.

## Main Functions

The Word Grouping Puzzle Game offers the following main functions:

- **Word Selection**: Users can select words from a list to form groups.
- **Group Validation**: After selecting four words, the game checks if they form a valid group.
- **Game Progression**: The game continues until all categories are correctly identified or the user runs out of tries.
- **User Feedback**: Provides immediate feedback on whether the selected group is correct or incorrect.
- **Graphical User Interface (GUI)**: An intuitive interface for easy interaction with the game.

## Installation

### Environment Setup

This application is developed in Python and does not require any external dependencies. Ensure you have Python installed on your system. You can download Python from the [official website](https://www.python.org/downloads/).

### Running the Game

1. **Clone the Repository**: Download or clone the repository containing the game files to your local machine.

2. **Navigate to the Directory**: Open a terminal or command prompt and navigate to the directory where the game files are located.

3. **Run the Game**: Execute the following command to start the game:
   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Launch the game by running `main.py`. A window will open displaying the list of words.

2. **Select Words**: Click on the words to select them. You need to select exactly four words to form a group. Selected words will appear sunken to indicate they are chosen.

3. **Check Group**: Once you have selected four words, click the "Check Group" button to validate your selection.

4. **Receive Feedback**: A message box will inform you whether your selected group is correct or incorrect. If correct, the words will be disabled, indicating they are part of a completed category.

5. **Continue Playing**: Repeat the process of selecting and checking groups until all categories are identified or you run out of tries.

6. **End Game**: The game ends when all categories are correctly identified or when you have exhausted the maximum number of tries. A message will display your result.

Enjoy the challenge and improve your categorization skills with the Word Grouping Puzzle Game!
