# Responsive Web-Based Calculator

Welcome to the user manual for the Responsive Web-Based Calculator. This document will guide you through the main functions of the calculator, how to set up the environment, and how to use the calculator effectively.

## Overview

The Responsive Web-Based Calculator is a simple yet powerful tool designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It features a clean and intuitive user interface that adapts to various screen sizes, making it accessible on both desktop and mobile devices.

## Main Functions

- **Numeric Keypad**: Allows users to input numbers from 0 to 9.
- **Basic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/).
- **Decimal Point**: Enables input of decimal numbers for more precise calculations.
- **Display Area**: Shows the current input and results of calculations.
- **Responsive Design**: Ensures the calculator is usable on different screen sizes, from mobile phones to desktop monitors.

## Installation

### Prerequisites

To use the calculator, you need a web browser that supports HTML, CSS, and JavaScript. No additional software or plugins are required.

### Quick Setup

1. **Download the Files**: Obtain the `index.html`, `styles.css`, and `script.js` files from the provided source.

2. **File Structure**: Ensure the files are in the same directory for proper linking:
   ```
   /calculator
   ├── index.html
   ├── styles.css
   └── script.js
   ```

3. **Open the Calculator**: Simply open the `index.html` file in your preferred web browser to start using the calculator.

## How to Use the Calculator

1. **Launch the Calculator**: Open the `index.html` file in a web browser.

2. **Input Numbers**: Click on the numeric buttons (0-9) to enter numbers into the display.

3. **Perform Operations**:
   - Click on an operator button (+, -, *, /) to perform the desired operation.
   - Enter the second number and click the `=` button to see the result.

4. **Decimal Numbers**: Use the `.` button to input decimal numbers.

5. **Clear and Reset**: To start a new calculation, simply begin entering new numbers or refresh the page.

6. **Error Handling**: The calculator will display "Error" if you attempt to divide by zero.

## Additional Information

- **Responsive Design**: The calculator's layout adjusts automatically to fit different screen sizes, ensuring usability on both mobile and desktop devices.
- **User Interaction**: Buttons provide visual feedback when clicked, enhancing the user experience.
