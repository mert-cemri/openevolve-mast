# Pong Game User Manual

Welcome to the Pong Game! This classic two-player game allows you to control a vertical paddle, moving it up and down to bounce a ball back to your opponent. The ball bounces between the two paddles and off the top and bottom edges. Players score a point when the opponent misses the ball. This manual will guide you through the installation, setup, and gameplay.

## Introduction

The Pong Game is a simple yet engaging application developed using Python and the Pygame library. It is designed to provide a fun and competitive experience for two players. The game features realistic ball physics, paddle control, and a scoring system.

## Installation

To run the Pong Game, you need to have Python and Pygame installed on your system. Follow the steps below to set up the environment:

### Step 1: Install Python

Ensure you have Python installed on your system. You can download it from the [official Python website](https://www.python.org/downloads/).

### Step 2: Install Pygame

Pygame is a set of Python modules designed for writing video games. Install Pygame using pip, the Python package manager, by running the following command in your terminal or command prompt:

```bash
pip install pygame
```

## Game Setup

Once you have Python and Pygame installed, follow these steps to set up the game:

1. Download the game files, including `main.py`, `paddle.py`, `ball.py`, and `score.py`.
2. Ensure all files are in the same directory.

## How to Play

1. **Start the Game**: Run the `main.py` file using Python. You can do this by navigating to the directory containing the game files and executing the following command:

   ```bash
   python main.py
   ```

2. **Control the Paddles**:
   - Player 1 controls the left paddle using the `W` key to move up and the `S` key to move down.
   - Player 2 controls the right paddle using the `UP` arrow key to move up and the `DOWN` arrow key to move down.

3. **Scoring**: Players score a point when the opponent misses the ball. The score is displayed at the top of the screen.

4. **End the Game**: To exit the game, close the game window or press `ESC`.

## Main Features

- **Two-Player Mode**: Enjoy a competitive game with a friend.
- **Realistic Ball Physics**: The ball bounces off paddles and edges, with speed increasing over time.
- **Scoring System**: Keep track of scores with a clear display at the top of the screen.
- **Simple Controls**: Easy-to-use keyboard controls for both players.
