# 2048 Game with 10x10 Grid

Welcome to the 2048 Game with a 10x10 grid! This game is a larger version of the classic 2048 puzzle game, where the objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

## Main Functions of the Software

- **Grid Management**: The game features a 10x10 grid where tiles can be moved and merged.
- **Tile Movement**: Players can move tiles up, down, left, or right. Tiles with the same number merge into one when they touch.
- **Score Tracking**: The game keeps track of the player's score, which increases with each successful merge.
- **Game Over Detection**: The game ends when no more moves are possible.

## Installation

To run the 2048 game, you need to have Python and the Pygame library installed on your system. Follow the steps below to set up the environment:

### Step 1: Install Python

Ensure that Python is installed on your system. You can download it from the [official Python website](https://www.python.org/downloads/).

### Step 2: Install Pygame

Pygame is required to run the graphical interface of the game. You can install it using pip:

```bash
pip install pygame
```

## How to Use/Play

### Step 1: Clone the Repository

Clone the repository containing the game code to your local machine.

### Step 2: Navigate to the Game Directory

Open a terminal or command prompt and navigate to the directory where the game files are located.

### Step 3: Run the Game

Execute the following command to start the game:

```bash
python main.py
```

### Step 4: Play the Game

- Use the arrow keys on your keyboard to move the tiles in the desired direction (up, down, left, or right).
- When two tiles with the same number touch, they merge into one.
- The game continues until no more moves are possible.
- Aim to create a tile with the number 2048 to win the game!

### Step 5: End of the Game

- The game will display "Game Over!" when no more moves are possible.
- You can close the game window to exit.

Enjoy playing the 2048 Game with a 10x10 grid and challenge yourself to reach the highest score possible!
