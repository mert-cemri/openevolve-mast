# Flappy Bird Clone User Manual

Welcome to the Flappy Bird Clone game! This manual will guide you through the installation process, introduce the main functions of the software, and provide instructions on how to play the game.

## Introduction

The Flappy Bird Clone is a simple yet addictive game where the player controls a bird, attempting to fly between sets of pipes without hitting them. The player presses a key to make the bird 'flap' and keep it from falling. Each successful pass through the pipes increases the player's score. The game ends if the bird collides with a pipe or the ground.

## Main Functions

- **Flap Control**: Press the spacebar to make the bird flap its wings and ascend.
- **Score Tracking**: The game tracks the player's score, which increases with each successful pass through the pipes.
- **Collision Detection**: The game ends if the bird collides with a pipe or the ground.
- **Random Pipe Generation**: Pipes are randomly generated with vertical gaps, providing a unique challenge each time you play.

## Installation

### Prerequisites

Before you can run the Flappy Bird Clone, you need to have Python and Pygame installed on your system.

1. **Python**: Ensure you have Python installed. You can download it from [python.org](https://www.python.org/downloads/).

2. **Pygame**: Install Pygame, a set of Python modules designed for writing video games. You can install it using pip:

   ```bash
   pip install pygame
   ```

### Download the Game

Clone or download the game files from the repository to your local machine.

### Running the Game

1. Navigate to the directory where the game files are located.

2. Run the main game file using Python:

   ```bash
   python main.py
   ```

## How to Play

- **Start the Game**: Run the `main.py` file to start the game.
- **Control the Bird**: Press the spacebar to make the bird flap its wings and ascend.
- **Avoid Obstacles**: Navigate the bird through the gaps in the pipes. Avoid hitting the pipes or the ground.
- **Score Points**: Each time you successfully pass through a set of pipes, your score increases.
- **Game Over**: The game ends if the bird collides with a pipe or the ground. You can restart the game by running the `main.py` file again.

## Conclusion

Enjoy playing the Flappy Bird Clone! Challenge yourself to achieve the highest score possible. If you encounter any issues or have questions, please refer to the documentation or reach out for support. Happy flapping!
