# Spelling Bee Puzzle Game

Welcome to the Spelling Bee Puzzle Game! This interactive application allows users to engage in a fun and challenging word-forming game using a set of given letters. The game is designed to enhance vocabulary and spelling skills while providing an enjoyable experience.

## Main Functions

- **Word Formation**: Form words using a central letter and six surrounding letters. Each word must include the central letter and be at least four letters long.
- **Scoring System**: Earn points based on the length of the words you form. Use all seven letters for a bonus score.
- **Feedback Mechanism**: Receive instant feedback on the validity of your words and your current score.

## Installation

### Prerequisites

- Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Quick Install

1. **Clone the Repository**: Download the source code from the repository or copy the provided code files (`main.py` and `game_logic.py`) into a directory on your computer.

2. **Install Dependencies**: This project does not require any external dependencies beyond Python's standard library. Simply ensure Python is installed.

3. **Run the Application**: Navigate to the directory containing `main.py` and execute the following command in your terminal or command prompt:

   ```bash
   python main.py
   ```

   This will launch the Spelling Bee Puzzle Game application.

## How to Play

1. **Launch the Game**: Run the application as described in the installation section. A window will open displaying the game interface.

2. **Understand the Instructions**: Read the instructions provided in the application window. You need to form words using the central letter and the surrounding letters. Each word must include the central letter and be at least four letters long.

3. **Form Words**: Enter your word guesses into the input field and click the "Submit" button. Your word will be checked for validity.

4. **Scoring**: Points are awarded based on the length of the word. If you use all seven letters, you receive a bonus score.

5. **Feedback**: After submitting a word, you will receive feedback indicating whether the word is valid and how many points it is worth. Your total score will be updated accordingly.

6. **Continue Playing**: Keep forming words and submitting them to increase your score. Enjoy the challenge of finding as many valid words as possible!

## Troubleshooting
