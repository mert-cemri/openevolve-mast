# Interactive Storytelling Game

Welcome to the Interactive Storytelling Game! This application allows players to immerse themselves in a narrative where their choices influence the story's progression and outcome. Players can explore different paths, build relationships, and collect items that will affect the story's ending.

## Main Functions

- **Narrative Segments**: The game presents various narrative segments to the player, each with a unique storyline and choices.
- **Branching Choices**: Players make decisions that lead to different story paths and outcomes.
- **Player State Tracking**: The game tracks player relationships and items collected, which influence the story's ending.
- **Multiple Endings**: Based on the player's choices, the game offers multiple possible endings.

## Installation

To run the Interactive Storytelling Game, you need to have Python installed on your system. Follow the steps below to set up the environment and dependencies:

### Step 1: Install Python

Ensure that Python is installed on your system. You can download it from the [official Python website](https://www.python.org/downloads/).

### Step 2: Set Up a Virtual Environment (Optional)

It's a good practice to create a virtual environment for your project to manage dependencies. You can create a virtual environment using the following commands:

```bash
# Create a virtual environment
python -m venv storytelling_env

# Activate the virtual environment
# On Windows
storytelling_env\Scripts\activate
# On macOS/Linux
source storytelling_env/bin/activate
```

### Step 3: Install Dependencies

Navigate to the project directory and install the required dependencies using pip:

```bash
pip install -r requirements.txt
```

*Note: Ensure that a `requirements.txt` file is present in the project directory with any necessary dependencies listed.*

## How to Play

1. **Start the Game**: Run the main script to start the game.

   ```bash
   python main.py
   ```

2. **Make Choices**: As the story unfolds, you will be presented with narrative segments and choices. Enter the number corresponding to your choice to proceed.

3. **Track Your Progress**: The game will keep track of your relationships and items collected throughout the journey.

4. **Reach an Ending**: Based on your decisions, the game will conclude with one of the multiple possible endings. Your relationships and items will be displayed at the end.

5. **Replay**: Feel free to replay the game to explore different paths and endings.

## Conclusion

The Interactive Storytelling Game offers a dynamic and engaging experience where your choices shape the narrative. Enjoy exploring the different paths and discovering the various endings! If you encounter any issues or have questions, please contact our support team for assistance.
