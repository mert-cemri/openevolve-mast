# Tic-Tac-Toe Game

Welcome to the Tic-Tac-Toe Game! This application allows two players to play the classic game of Tic-Tac-Toe with a user-friendly graphical interface.

## Main Functions

- **Two-Player Game**: The game allows two players to take turns playing as 'X' and 'O'.
- **Winner Determination**: The game automatically checks for a winner after each move and announces the result.
- **Draw Detection**: If the board is full and there is no winner, the game declares a draw.
- **Graphical User Interface (GUI)**: The game features a simple and intuitive GUI built using the Tkinter library.
- **Restart Functionality**: Players can restart the game at any time to play again.

## Quick Install

### Prerequisites

- Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/).

### Installation

1. Clone the repository or download the source code files.
2. Navigate to the directory containing the source code files.

Since there are no external dependencies required, you can directly run the game without additional installations.

## How to Play

1. **Run the Game**: Open a terminal or command prompt, navigate to the directory containing the source code, and execute the following command:
   ```bash
   python main.py
   ```

2. **Game Interface**: A window will open displaying the Tic-Tac-Toe board with 9 buttons representing the 3x3 grid.

3. **Make a Move**: Players take turns clicking on the buttons to place their symbol ('X' or 'O') on the board.

4. **Check Results**: After each move, the game checks for a winner or a draw. If a player wins, a message will be displayed, and the board will be disabled.

5. **Restart the Game**: To play again, click the 'Restart' button to reset the board and start a new game.
