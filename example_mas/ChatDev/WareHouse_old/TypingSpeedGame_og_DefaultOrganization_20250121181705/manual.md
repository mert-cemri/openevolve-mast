# Typing Practice Game

Welcome to the Typing Practice Game! This application is designed to help users improve their typing speed and accuracy through engaging practice sessions. The game displays random words or phrases, and users are challenged to type them as quickly and accurately as possible within a set time limit.

## Main Functions

- **Random Word Display**: The game displays random words from a predefined list for the user to type.
- **Typing Speed Measurement**: Calculates the user's typing speed in words per minute (WPM).
- **Accuracy Calculation**: Measures the accuracy of the user's typing by comparing typed words with displayed words.
- **Results Display**: At the end of the session, the game shows the user's WPM and accuracy percentage.

## Installation

This project does not require any external dependencies beyond Python's standard library. Ensure you have Python installed on your system. You can download Python from the [official website](https://www.python.org/downloads/).

1. **Clone the Repository**: Download the project files to your local machine.

   ```bash
   git clone <repository-url>
   cd typing-practice-game
   ```

2. **Run the Application**: Execute the main application file to start the game.

   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Launch the application by running `main.py`. A window will open with a "Start" button.

2. **Begin Typing**: Click the "Start" button to begin the game. A word will appear on the screen. Type the word in the text entry field and press "Enter" to submit.

3. **Continue Typing**: New words will appear as you type correctly. Continue typing as many words as possible within the 60-second time limit.

4. **View Results**: After the time is up, a message box will display your typing speed (WPM) and accuracy percentage.
