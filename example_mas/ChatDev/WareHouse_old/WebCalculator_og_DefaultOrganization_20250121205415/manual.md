# Responsive Web-Based Calculator

This user manual provides a comprehensive guide on how to use the Responsive Web-Based Calculator, including its main functions, installation of environment dependencies, and instructions on how to use the calculator.

## Main Functions

The Responsive Web-Based Calculator is a simple, intuitive tool designed to perform basic arithmetic operations. It includes the following features:

- **Numeric Keypad**: Allows users to input numbers from 0 to 9.
- **Basic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/).
- **Display Area**: Shows the current input and results of calculations.
- **Clear Function**: Resets the calculator to its initial state.
- **Responsive Design**: Ensures usability across various screen sizes, from desktops to mobile devices.

## Installation

To use the calculator, you need to have a basic web environment set up. Follow these steps to get started:

1. **Download the Files**: Ensure you have the following files in your project directory:
   - `index.html`: Contains the HTML structure of the calculator.
   - `styles.css`: Provides the styling for the calculator.
   - `script.js`: Includes the JavaScript logic for calculator operations.

2. **Set Up a Local Server** (optional): For a better development experience, you might want to set up a local server. You can use tools like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for Visual Studio Code or [http-server](https://www.npmjs.com/package/http-server) for Node.js.

3. **Open the HTML File**: You can simply open the `index.html` file in your web browser to start using the calculator.

## How to Use the Calculator

1. **Launch the Calculator**: Open the `index.html` file in your web browser. You will see the calculator interface with a display area and buttons.

2. **Input Numbers**: Click on the numeric buttons (0-9) to input numbers. The numbers will appear in the display area.

3. **Perform Operations**:
   - Click on the operation buttons (+, -, *, /) to choose the desired arithmetic operation.
   - Input the second number.
   - Click the `=` button to compute the result, which will be displayed in the display area.

4. **Clear the Display**: Click the `C` button to reset the calculator and clear the display area.

5. **Responsive Design**: The calculator is designed to adjust its layout based on the screen size, ensuring a seamless experience on both desktop and mobile devices.

## Troubleshooting

- **Error Handling**: If you attempt to divide by zero, the calculator will display an "Error" message. Clear the display to reset the calculator.

- **Display Issues**: If the calculator does not display correctly, ensure that all files (`index.html`, `styles.css`, and `script.js`) are correctly linked and loaded in your browser.

By following this manual, you should be able to effectively use the Responsive Web-Based Calculator for your basic arithmetic needs. If you encounter any issues or have further questions, please feel free to reach out for support.
