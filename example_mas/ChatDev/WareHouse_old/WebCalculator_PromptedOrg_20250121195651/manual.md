# Responsive Web-Based Calculator

A simple, responsive web-based calculator application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator features a numeric keypad and a display area to show results, with a clean and intuitive user interface that works seamlessly on various screen sizes.

## Main Functions

- **Numeric Keypad**: Allows users to input numbers and decimal points.
- **Basic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/).
- **Display Area**: Shows the current input and results of calculations.
- **Clear Function**: Resets the calculator to its initial state.
- **Responsive Design**: Ensures usability across different devices and screen sizes.

## Installation

This project is a web-based application and does not require any external Python dependencies. It is built using HTML, CSS, and JavaScript.

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Safari, Edge).

### Steps to Install and Run

1. **Download the Project Files**: Ensure you have all the necessary files (`index.html`, `styles.css`, `calculator.js`, and `app.js`) in a single directory.

2. **Open the Application**:
   - Locate the `index.html` file in your project directory.
   - Double-click the `index.html` file to open it in your default web browser.

3. **Start Using the Calculator**:
   - The calculator interface will appear in your browser.
   - Use the numeric keypad to input numbers.
   - Click on the operation buttons (+, -, *, /) to perform calculations.
   - Press the "=" button to display the result.
   - Use the "C" button to clear the display and reset the calculator.

## Usage Instructions

- **Entering Numbers**: Click on the numeric buttons (0-9) to enter numbers. Use the "." button to add a decimal point.
- **Performing Calculations**:
  - Enter the first number.
  - Click on an operation button (+, -, *, /).
  - Enter the second number.
  - Click the "=" button to see the result.
- **Clearing the Display**: Click the "C" button to clear the current input and reset the calculator.

## Troubleshooting

- **Division by Zero**: If you attempt to divide by zero, an error message will alert you, and the calculator will reset.
- **Responsive Issues**: Ensure your browser window is not minimized to an extremely small size to maintain the calculator's usability.
