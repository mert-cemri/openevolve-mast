# Rock-Paper-Scissors Game User Manual

Welcome to the Rock-Paper-Scissors game! This simple yet entertaining game allows you to play the classic game of Rock-Paper-Scissors against the computer. Follow this manual to get started and enjoy the game.

## Main Functions

The Rock-Paper-Scissors game includes the following main functions:

1. **User Choice**: The user selects one of the three options: Rock, Paper, or Scissors.
2. **Computer Choice**: The computer randomly selects one of the three options.
3. **Determine Winner**: The game compares the user's choice and the computer's choice to determine the winner based on the classic rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. **Display Result**: The game displays the choices made by both the user and the computer, and announces the winner.
5. **Replay Option**: After each round, the user is prompted to play again or exit the game.

## Installation

To play the Rock-Paper-Scissors game, you need to have Python installed on your system. Follow these steps to set up the environment and run the game:

### Step 1: Install Python

Ensure that Python is installed on your system. You can download and install Python from the official website: [Python.org](https://www.python.org/downloads/).

### Step 2: Download the Game Code

Download the `main.py` file containing the game code. Save it to a directory of your choice.

### Step 3: Run the Game

Open a terminal or command prompt, navigate to the directory where you saved `main.py`, and run the following command:

```bash
python main.py
```

This command will start the Rock-Paper-Scissors game.

## How to Play

1. **Start the Game**: Run the `main.py` file as described in the installation section.
2. **Make Your Choice**: When prompted, enter your choice of "rock", "paper", or "scissors".
3. **View Results**: The game will display your choice, the computer's choice, and the result of the round.
4. **Replay or Exit**: After each round, you will be asked if you want to play again. Enter "yes" to play another round or "no" to exit the game.
