# Space Invaders Game

Welcome to the Space Invaders Game! This is a simplified version of the classic arcade game where you control a ship at the bottom of the screen. Your mission is to destroy the descending alien rows before they reach the bottom. The game ends if the aliens reach the bottom or you defeat all the aliens.

## Main Functions

- **Player Control**: Move your ship horizontally across the bottom of the screen using the arrow keys and fire shots using the spacebar.
- **Alien Invasion**: Aliens descend from the top of the screen. Your goal is to shoot them before they reach the bottom.
- **Game Over Conditions**: The game ends if any alien reaches the bottom of the screen or if you successfully destroy all aliens.

## Installation

### Prerequisites

To run the Space Invaders game, you need to have Python and Pygame installed on your system. Follow the steps below to set up the environment:

1. **Install Python**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

2. **Install Pygame**: Pygame is a set of Python modules designed for writing video games. You can install it using pip:

   ```bash
   pip install pygame
   ```

### Download the Game

Clone the repository or download the game files to your local machine.

```bash
git clone <repository-url>
```

Replace `<repository-url>` with the actual URL of the repository containing the game files.

## How to Play

1. **Navigate to the Game Directory**: Open a terminal or command prompt and navigate to the directory where the game files are located.

   ```bash
   cd path/to/game-directory
   ```

2. **Run the Game**: Execute the main script to start the game.

   ```bash
   python main.py
   ```

3. **Game Controls**:
   - Use the **Left Arrow Key** to move the ship left.
   - Use the **Right Arrow Key** to move the ship right.
   - Press the **Spacebar** to shoot bullets at the aliens.

4. **Objective**: Destroy all the aliens before they reach the bottom of the screen. The game ends if an alien reaches the bottom or all aliens are destroyed.

5. **Exit the Game**: Close the game window or press `Ctrl + C` in the terminal to exit the game.

## Additional Information

- The game runs at 60 frames per second for smooth gameplay.
- The player's ship is represented by a green rectangle, bullets by white rectangles, and aliens by red rectangles.
- The game window size is 800x600 pixels.

Enjoy playing Space Invaders and good luck defending the Earth from alien invaders!
