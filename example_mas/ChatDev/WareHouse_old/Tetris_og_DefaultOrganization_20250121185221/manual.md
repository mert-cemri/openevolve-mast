# Tetris Game User Manual

Welcome to the Tetris Game! This manual will guide you through the installation process, introduce the main features of the game, and provide instructions on how to play.

## Introduction

The Tetris Game is a classic puzzle game where tetrominoes fall from the top of the screen. The player can move and rotate these pieces to fit them into complete lines. Completed lines are cleared, and the player's score increases accordingly. The game ends when no more pieces can be placed on the board.

## Main Features

- **Interactive Gameplay:** Move and rotate tetrominoes using keyboard controls.
- **Scoring System:** Earn points by clearing lines.
- **Dynamic Board Updates:** The board updates in real-time as pieces are moved or dropped.
- **Game Over Condition:** The game ends when the board is full and no more pieces can be placed.

## Installation

To play the Tetris Game, you need to have Python and the Pygame library installed on your system. Follow the steps below to set up the environment:

### Step 1: Install Python

Ensure that Python is installed on your system. You can download it from the official [Python website](https://www.python.org/downloads/).

### Step 2: Install Pygame

Pygame is a set of Python modules designed for writing video games. You can install it using pip:

```bash
pip install pygame==2.1.2
```

Alternatively, you can use a `requirements.txt` file to install dependencies:

```bash
pip install -r requirements.txt
```

## How to Play

### Starting the Game

1. Open a terminal or command prompt.
2. Navigate to the directory where the game files are located.
3. Run the following command to start the game:

```bash
python main.py
```

### Controls

- **Left Arrow Key:** Move the tetromino left.
- **Right Arrow Key:** Move the tetromino right.
- **Down Arrow Key:** Move the tetromino down faster.
- **Up Arrow Key:** Rotate the tetromino.

### Objective

The objective of the game is to clear as many lines as possible by fitting the falling tetrominoes into complete rows. Each cleared line increases your score. The game ends when the board is full and no more pieces can be placed.

### Scoring

- Each line cleared adds 100 points to your score.
- The score is displayed on the screen during gameplay.
