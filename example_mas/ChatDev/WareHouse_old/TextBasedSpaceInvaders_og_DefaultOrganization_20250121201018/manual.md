# Space Invaders Game

A simplified version of the classic Space Invaders game where the player controls a ship at the bottom of the screen, moving horizontally and firing shots to destroy descending alien rows. The game ends if aliens reach the bottom or the player defeats all aliens.

## Quick Install

To get started with the Space Invaders game, you need to install the required dependencies. The game is built using Python and the Pygame library.

### Prerequisites

- Python 3.x installed on your system.
- Pygame library.

### Installation

1. **Clone the Repository**

   Clone the project repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

   Replace `<repository-url>` with the actual URL of the repository.

2. **Navigate to the Project Directory**

   Change your directory to the project folder:

   ```bash
   cd <project-directory>
   ```

   Replace `<project-directory>` with the actual directory name.

3. **Install Dependencies**

   Install the required dependencies using pip:

   ```bash
   pip install -r requirements.txt
   ```

   This command will install the Pygame library, which is necessary to run the game.

## ðŸ¤” What is this?

The Space Invaders game is a simple yet engaging game where the player controls a spaceship at the bottom of the screen. The main objective is to shoot and destroy all the aliens descending from the top before they reach the bottom of the screen.

### Main Features

- **Player Control:** Move the player's ship left and right using the arrow keys and fire shots using the spacebar.
- **Alien Movement:** Aliens move horizontally and descend gradually. The game ends if any alien reaches the bottom.
- **Shooting Mechanism:** The player can shoot bullets to destroy aliens. Successful hits remove aliens from the screen.
- **Game Over Conditions:** The game ends when all aliens are destroyed or if any alien reaches the bottom of the screen.

## ðŸ“– How to Play

1. **Start the Game**

   Run the main script to start the game:

   ```bash
   python main.py
   ```

2. **Game Controls**

   - **Move Left:** Press the left arrow key to move the player's ship to the left.
   - **Move Right:** Press the right arrow key to move the player's ship to the right.
   - **Shoot:** Press the spacebar to fire a shot from the player's ship.

3. **Objective**

   - Destroy all the aliens before they reach the bottom of the screen.
   - Avoid letting any alien reach the bottom to prevent game over.

4. **End of Game**

   - The game ends when all aliens are destroyed or if an alien reaches the bottom of the screen.

Enjoy playing the Space Invaders game and aim for the highest score by defeating all the aliens!
