# Gomoku Game User Manual

## Introduction
Welcome to the Gomoku game! This user manual will guide you through the installation process and explain how to play the game.

## Installation
To install the Gomoku game, please follow the steps below:

1. Make sure you have Python installed on your computer. If not, you can download it from the official Python website (https://www.python.org/downloads/).

2. Open a terminal or command prompt.

3. Navigate to the directory where you have saved the `main.py` and `game.py` files.

4. Install the required dependencies by running the following command:
   ```
   pip install -r requirements.txt
   ```

## How to Play
Once you have installed the game, you can start playing Gomoku by following these steps:

1. Open a terminal or command prompt.

2. Navigate to the directory where you have saved the `main.py` and `game.py` files.

3. Run the following command to start the game:
   ```
   python main.py
   ```

4. A new window will open with the Gomoku game board.

5. The game board consists of a 15x15 grid. Each cell represents a position on the board.

6. Players take turns placing their pieces on the board. The first player is 'X' and the second player is 'O'.

7. To make a move, click on an empty cell on the board. The cell will be filled with the current player's piece.

8. The game will automatically check for a winner after each move. If a player has five consecutive pieces in a row, column, or diagonal, they win the game.

9. If a player wins, a message box will appear displaying the winner's name.

10. To reset the game and start a new round, click the "Reset" button.

11. You can close the game window at any time to exit the game.

## Conclusion
Congratulations! You have successfully installed and learned how to play the Gomoku game. Enjoy playing and have fun! If you have any questions or encounter any issues, please don't hesitate to contact our support team.
