# Classic Snake Game
## Description
This is a classic Snake game implemented in Python. The player controls a snake using directional inputs to move around the board and eat food items. Each time the snake eats, it grows in length. The game ends if the snake collides with itself or the boundary. The board updates after each move, and the current score is displayed based on food eaten. The game offers multiple difficulty levels: easy, medium, and hard.
## Requirements
- Python 3.x
- curses library (built-in on Unix-based systems)
