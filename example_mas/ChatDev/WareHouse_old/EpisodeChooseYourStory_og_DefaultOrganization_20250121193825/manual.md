# Interactive Storytelling Game

Welcome to the Interactive Storytelling Game! This game offers an immersive experience where your decisions shape the narrative. Explore different paths, make choices, and see how your actions influence the story's outcome.

## Main Features

- **Narrative Segments**: Experience a rich story divided into segments, each presenting unique scenarios.
- **Branching Choices**: Make decisions that lead to different story paths and outcomes.
- **Dynamic Variables**: Track variables such as relationships or items that influence future events and endings.
- **Graphical User Interface**: Enjoy an intuitive and interactive GUI built with Tkinter.

## Installation Guide

### Prerequisites

- **Python**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Installation Steps

1. **Clone the Repository**: Clone the game repository to your local machine using the following command:
   ```bash
   git clone <repository-url>
   ```
   Replace `<repository-url>` with the actual URL of the repository.

2. **Navigate to the Project Directory**: Change your directory to the project folder:
   ```bash
   cd <project-directory>
   ```
   Replace `<project-directory>` with the name of the cloned directory.

3. **Install Dependencies**: The game uses Tkinter, which is part of the Python standard library, so no additional installations are required. However, ensure your Python environment is set up correctly.

### Running the Game

1. **Execute the Main Script**: Run the main script to start the game:
   ```bash
   python main.py
   ```

2. **Enjoy the Game**: The game window will open, and you can start playing by making choices and exploring the narrative.

## How to Play

- **Start the Game**: Upon launching, the game presents the first narrative segment.
- **Make Choices**: Read the narrative and select from available choices. Each choice will lead to a different segment and may affect game variables.
- **Track Your Progress**: The game tracks your decisions and variables, influencing future story events and potential endings.
- **Explore Different Paths**: Replay the game to explore different choices and discover various storylines and endings.

## Troubleshooting

- **Display Issues**: If you encounter issues with the GUI display, ensure your system supports Tkinter and that your Python environment is correctly configured.
- **Game Crashes**: If the game crashes, check for any error messages in the terminal and ensure all files are correctly placed in the project directory.
