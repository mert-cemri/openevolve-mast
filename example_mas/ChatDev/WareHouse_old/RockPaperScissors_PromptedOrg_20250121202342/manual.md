# Rock-Paper-Scissors Game

Welcome to the Rock-Paper-Scissors game! This is a simple yet entertaining game where you can challenge the computer and see who wins. The game follows the traditional rules of Rock-Paper-Scissors:

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Quick Install

To play the Rock-Paper-Scissors game, you need to have Python installed on your system. If you haven't installed Python yet, you can download it from [python.org](https://www.python.org/downloads/).

### Installing Dependencies

This game requires the `random` module, which is included in Python's standard library. Therefore, you don't need to install any additional packages.

## How to Play

1. **Download the Game Code:**

   Save the provided code in a file named `main.py`.

2. **Run the Game:**

   Open your terminal or command prompt and navigate to the directory where `main.py` is located. Run the following command:

   ```bash
   python main.py
   ```

3. **Gameplay Instructions:**

   - When prompted, enter your choice: "rock", "paper", or "scissors".
   - The computer will randomly select its choice.
   - The game will display the computer's choice and announce the winner.
   - You will be asked if you want to play again. Enter "yes" to replay or "no" to exit the game.

4. **End of Game:**

   If you choose not to replay, the game will thank you for playing and exit.
