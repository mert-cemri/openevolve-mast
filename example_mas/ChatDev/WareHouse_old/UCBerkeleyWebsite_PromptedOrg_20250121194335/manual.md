## Main Functions

The UC Berkeley website prototype includes the following main functions:

- **Responsive Design**: The website adjusts its layout based on the screen size, ensuring a seamless experience on both desktop and mobile devices.
- **Navigation**: A main navigation menu allows users to easily access major sections such as Admissions, Academics, Research, and News & Events.
- **Dynamic Content Loading**: The News & Events section dynamically loads content to keep users updated with the latest information.
- **Accessibility Features**: The website includes features to enhance accessibility, such as keyboard navigation and ARIA roles.

## Installation and Environment Setup

To run the UC Berkeley website prototype, follow these steps:

1. **Clone the Repository**: Download the source code from the repository to your local machine.

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**: Change your working directory to the project folder.

   ```bash
   cd <project-directory>
   ```

3. **Install Dependencies**: Ensure you have a modern web browser installed (e.g., Google Chrome, Mozilla Firefox) to view the website. No additional software installations are required as the prototype runs in a web browser.

## How to Use the Prototype

1. **Open the Website**: Open the `index.html` file in your web browser to view the UC Berkeley landing page.

2. **Navigate the Website**: Use the navigation menu at the top of the page to access different sections:
   - **Admissions**: Information about the admissions process.
   - **Academics**: Details about academic programs and offerings.
   - **Research**: Insights into research initiatives and projects.
   - **News & Events**: Latest news and upcoming events at UC Berkeley.

3. **Responsive Design**: Resize your browser window to see how the website adapts to different screen sizes. On smaller screens, the navigation menu will collapse into a hamburger menu for easier access.

4. **Accessibility Features**: Use the keyboard to navigate through the website. Focus outlines will appear on links when using the Tab key, enhancing visibility for keyboard users.

5. **Dynamic Content**: The News & Events section will automatically load additional content after a short delay, simulating real-time updates.
