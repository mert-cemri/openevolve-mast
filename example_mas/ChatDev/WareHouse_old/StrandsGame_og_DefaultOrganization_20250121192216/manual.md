# Strands Game

Welcome to the Strands Game! This is an engaging word-segmentation puzzle where players combine strands of text to form meaningful words or phrases. The game verifies valid strand formations and confirms completion once all strands are correctly merged.

## Main Functions

- **Strand Selection**: Players can select strands of text to combine them.
- **Validation**: The game checks if the combined strands form valid words or phrases.
- **Feedback**: Provides immediate feedback on whether the combination is correct or not.
- **Reset**: Allows players to reset their selections and try again.

## Installation

To run the Strands Game, you need to have Python installed on your system. Follow these steps to set up your environment:

1. **Clone the Repository**: Download the game files from the repository.

2. **Install Dependencies**: The game uses the `tkinter` library for the graphical user interface, which is included with Python. Ensure your Python environment is set up correctly.

3. **Run the Game**: Navigate to the directory containing the game files and execute the following command in your terminal or command prompt:

   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Launch the game by running the `main.py` file. A window will open displaying the available strands.

2. **Select Strands**: Click on the buttons representing the strands of text you want to combine. Your selections will appear in the display area.

3. **Check Your Solution**: Once you have combined the strands, click the "Check" button to see if your combination forms valid words or phrases.

4. **Feedback**: If your combination is correct, a "Correct!" message will appear. If not, you'll see "Try Again!".

5. **Reset**: If you want to start over, click the "Reset" button to clear your selections and try again.

## Additional Information

- **Customization**: You can modify the strands and valid words by editing the `main.py` and `validator.py` files. This allows you to create new puzzles and challenges.

- **Troubleshooting**: If you encounter issues starting the GUI, ensure you have a display environment set up, especially if you're running the game on a server or headless system.

Enjoy playing the Strands Game and challenge your word segmentation skills!
