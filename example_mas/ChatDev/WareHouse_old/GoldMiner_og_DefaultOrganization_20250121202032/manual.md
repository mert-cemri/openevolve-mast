# Gold Miner Game

Welcome to the Gold Miner Game! This is an engaging and interactive game where players control a claw to collect gold and other objects. The objective is to gather enough gold before the time runs out. This manual will guide you through the installation, setup, and gameplay.

## Quick Install

To get started with the Gold Miner Game, you need to install the required dependencies. The game is developed using Python and the Pygame library.

### Environment Setup

1. **Python Installation**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

2. **Pygame Installation**: Install the Pygame library using pip. Open your terminal or command prompt and run the following command:

   ```bash
   pip install pygame==2.1.2
   ```

   This will install the specific version of Pygame required for the game.

### Running the Game

1. **Clone the Repository**: Download or clone the game repository to your local machine.

2. **Navigate to the Game Directory**: Open your terminal or command prompt and navigate to the directory where the game files are located.

3. **Start the Game**: Run the following command to start the game:

   ```bash
   python main.py
   ```

   This will launch the game window.

## ðŸ¤” What is this Game?

The Gold Miner Game is a fun and challenging game where you control a claw that moves back and forth. Your goal is to time your grabs to collect gold and other objects. Each object has a value and takes time to reel in. The level ends when the time runs out or when you meet the minimum gold value required.

### Main Features

- **Claw Control**: Use the spacebar to grab objects as the claw moves back and forth.
- **Object Collection**: Collect various objects, each with different values and weights.
- **Level Completion**: Complete the level by collecting the minimum required gold before the time runs out.
- **Visual Feedback**: The game displays the position of the claw and objects, updating after each grab.

## ðŸ“– How to Play

1. **Start the Game**: Launch the game using the command provided above.

2. **Control the Claw**: The claw will automatically move back and forth. Press the spacebar to grab objects when the claw is positioned over them.

4. **Complete the Level**: Gather enough gold to meet the level's minimum requirement before the time runs out.

5. **Enjoy and Repeat**: Try to beat your previous scores and enjoy the challenge!
