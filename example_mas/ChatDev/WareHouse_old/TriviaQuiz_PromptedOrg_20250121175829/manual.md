Welcome to the Trivia Quiz Program! This application is designed to provide an engaging and educational experience through a series of multiple-choice and short-answer questions on various topics. This manual will guide you through the installation process, introduce the main functions of the software, and explain how to use and play the quiz.

## Introduction

The Trivia Quiz Program is a Python-based application that allows users to test their knowledge on various topics. The program presents questions, collects user answers, tracks the score, and displays the total correct answers at the end of the quiz.

## Installation

To run the Trivia Quiz Program, you need to have Python installed on your system. Follow these steps to set up the environment and run the application:

1. **Install Python**: If you don't have Python installed, download and install it from [python.org](https://www.python.org/downloads/).

2. **Clone the Repository**: Clone the repository containing the Trivia Quiz Program code to your local machine.

   ```bash
   git clone <repository-url>
   ```

3. **Navigate to the Project Directory**: Change into the directory where the code is located.

   ```bash
   cd <repository-directory>
   ```

4. **Install Dependencies**: The program does not require any external Python packages, so no additional dependencies need to be installed.

## Main Functions

The Trivia Quiz Program consists of several key components:

- **Question Management**: The program manages a list of questions, each with multiple-choice options and a correct answer.

- **User Interaction**: The program presents questions to the user, collects answers, and provides feedback on whether the answer was correct or not.

- **Score Tracking**: The program tracks the user's score throughout the quiz and displays the final score at the end.

## How to Play

1. **Start the Quiz**: Run the `main.py` file to start the quiz.

   ```bash
   python main.py
   ```

2. **Answer Questions**: The program will present each question with multiple-choice options. Enter the number corresponding to your chosen answer and press Enter.

3. **Receive Feedback**: After each answer, the program will inform you if your answer was correct or incorrect and provide the correct answer if necessary.

4. **View Final Score**: Once all questions have been answered, the program will display your total score and the number of correct answers.

## Troubleshooting

- **Python Not Installed**: Ensure that Python is installed and added to your system's PATH.

- **Incorrect Answers**: Double-check your input to ensure you are entering the correct number corresponding to your chosen answer.

- **Program Errors**: If you encounter any errors while running the program, ensure that all files (`main.py`, `quiz.py`, `question.py`, `quiz_interface.py`) are present in the same directory and have not been modified.
