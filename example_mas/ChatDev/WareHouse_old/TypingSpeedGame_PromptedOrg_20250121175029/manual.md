# Typing Practice Game

Welcome to the Typing Practice Game! This application is designed to help you improve your typing speed and accuracy through engaging practice sessions. The game displays random phrases for you to type, measures your performance in words per minute (WPM), and calculates your typing accuracy.

## Main Functions

- **Random Phrase Display**: The game provides a random phrase for you to type, ensuring a varied and challenging experience each time you play.
- **Typing Speed Measurement**: Your typing speed is calculated in words per minute (WPM), allowing you to track your progress over time.
- **Accuracy Calculation**: The game evaluates your typing accuracy as a percentage, helping you identify areas for improvement.
- **Real-Time Feedback**: At the end of each session, the game displays your WPM and accuracy, providing immediate feedback on your performance.

## Installation

### Environment Setup

The Typing Practice Game is built using Python and does not require any external dependencies. To get started, ensure you have Python installed on your system. You can download Python from the official website: [python.org](https://www.python.org/).

### Quick Install

1. **Clone the Repository**: Download the source code from the repository or copy the provided code files (`main.py`, `typing_game.py`, `phrase_provider.py`, `timer.py`) into a directory on your local machine.

2. **Navigate to the Directory**: Open a terminal or command prompt and navigate to the directory containing the code files.

3. **Run the Game**: Execute the following command to start the game:
   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Run the `main.py` script to launch the Typing Practice Game. You will be greeted with a welcome message.

2. **Type the Displayed Phrase**: The game will display a random phrase. Your task is to type the phrase as accurately and quickly as possible.

3. **Submit Your Input**: After typing the phrase, press Enter to submit your input.

4. **View Your Results**: The game will calculate and display your words per minute (WPM) and accuracy percentage. Use this feedback to assess your typing skills.

5. **Repeat**: Feel free to play multiple rounds to improve your typing speed and accuracy.

## Additional Information

- **Phrases**: The game includes a set of predefined phrases that are randomly selected for each session. These phrases are designed to challenge your typing skills with varying lengths and complexities.

- **Timing**: The game uses a timer to measure the time taken to type each phrase, which is crucial for calculating your WPM.

- **Feedback**: The feedback provided at the end of each session is intended to help you track your progress and identify areas for improvement.

Enjoy playing the Typing Practice Game and watch your typing skills improve over time!
