# Typing Practice Game

A simple and interactive typing practice game that measures your typing speed and accuracy. The game displays random words or phrases, and you type them within a set time. It calculates words-per-minute (WPM) and accuracy percentage, displaying the final results at the end.

## Quick Install

To run the Typing Practice Game, you need to have Python installed on your system. You can download and install Python from [python.org](https://www.python.org/downloads/).

### Step 1: Clone the Repository

Clone the repository to your local machine using the following command:

```bash
git clone <repository-url>
```

Replace `<repository-url>` with the actual URL of the repository.

### Step 2: Install Dependencies

Navigate to the project directory and install the required dependencies. This project does not have any external dependencies, so you can skip this step.

### Step 3: Run the Game

To start the game, navigate to the project directory and run the following command:

```bash
python main.py
```

The Typing Practice Game is designed to help users improve their typing speed and accuracy. It provides a fun and engaging way to practice typing by displaying random phrases that the user needs to type as quickly and accurately as possible.

### Main Functions

- **Random Phrase Display**: The game displays a random phrase from a predefined list for the user to type.
- **Timing**: The game measures the time taken by the user to type the phrase.
- **WPM Calculation**: Calculates the words-per-minute based on the time taken and the number of words typed.
- **Accuracy Calculation**: Calculates the accuracy percentage by comparing the user's input with the displayed phrase.
- **Result Display**: Displays the calculated WPM and accuracy percentage at the end of the game.

1. **Start the Game**: Run the `main.py` file to start the game.
2. **Type the Displayed Phrase**: The game will display a random phrase. Type the phrase as quickly and accurately as you can.

4. **View Results**: The game will display your typing speed in words-per-minute (WPM) and your accuracy percentage.
5. **Repeat**: You can play the game as many times as you like to improve your typing skills.
