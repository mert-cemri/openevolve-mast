# Trivia Quiz Program

Welcome to the Trivia Quiz Program! This application allows users to test their knowledge on various topics through multiple-choice and short-answer questions. The program tracks the user's score and displays the total correct answers at the end of the quiz.

## Main Functions

- **Question Management**: Create and manage trivia questions with multiple-choice or short-answer formats.
- **Quiz Conduction**: Present questions to the user, accept answers, and provide immediate feedback on correctness.
- **Score Tracking**: Keep track of the user's score throughout the quiz and display the final score at the end.

## Quick Install

To get started with the Trivia Quiz Program, you need to have Python installed on your system. The program requires Python version 3.6 or higher.

### Installation Steps

1. **Clone the Repository**: Download the source code from the repository.
   ```bash
   git clone <repository-url>
   cd trivia-quiz-program
   ```

2. **Set Up the Environment**: Ensure you have Python 3.6 or higher installed. You can check your Python version with:
   ```bash
   python --version
   ```

3. **Install Dependencies**: The program does not require any external dependencies, but it's good practice to create a virtual environment.
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

4. **Run the Program**: Execute the main script to start the quiz.
   ```bash
   python main.py
   ```

## How to Use/Play

1. **Start the Quiz**: Run the `main.py` script to begin the quiz. You will be presented with a series of questions.

2. **Answer Questions**: For multiple-choice questions, enter the number corresponding to your chosen answer. For short-answer questions, type your answer and press Enter.

3. **Receive Feedback**: After each question, you will be informed whether your answer was correct or incorrect, along with the correct answer if applicable.

4. **View Final Score**: At the end of the quiz, your total score will be displayed, showing the number of correct answers out of the total questions.

Enjoy testing your knowledge and learning new facts with the Trivia Quiz Program!
