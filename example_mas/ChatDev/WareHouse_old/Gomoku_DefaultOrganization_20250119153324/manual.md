# Gomoku Game User Manual

## Introduction

Welcome to the Gomoku game! Gomoku is a classic board game played on a 10x10 grid. The objective of the game is to be the first player to form a line of five consecutive stones horizontally, vertically, or diagonally.

This user manual will guide you through the installation process and explain how to play the game.

## Installation

To play the Gomoku game, you need to install the following dependencies:

- Python 3.x
- Tkinter library

Please follow the steps below to install the dependencies:

1. Install Python 3.x: Visit the official Python website (https://www.python.org) and download the latest version of Python for your operating system. Follow the installation instructions provided.

2. Install Tkinter library: Open a terminal or command prompt and run the following command:

   ```
   pip install tkinter
   ```

   If you are using a virtual environment, make sure it is activated before running the command.

## Running the Game

Once you have installed the dependencies, you can run the Gomoku game by following these steps:

1. Download the game files: Download the `main.py` and `game.py` files provided.

2. Open a terminal or command prompt and navigate to the directory where you saved the game files.

3. Run the following command to start the game:

   ```
   python main.py
   ```

   The game window will open, and you can start playing.

## Game Instructions

1. The game window will display a 10x10 grid.

2. Players take turns placing their stones on the grid. The first player is represented by black stones, and the second player is represented by white stones.

3. To make a move, click on an empty cell on the grid. The stone will be placed in that cell, and it will be the next player's turn.

4. The game will automatically detect when a player forms a line of five consecutive stones and declare that player as the winner.

5. If the grid is filled with stones and no player has formed a line of five, the game will end in a draw.

6. To start a new game, close the game window and run the `main.py` file again.

## Conclusion
