# Typing Practice Game

Welcome to the Typing Practice Game! This application is designed to help you improve your typing speed and accuracy by providing a fun and interactive way to practice. The game displays random words or phrases, and your task is to type them as quickly and accurately as possible. At the end of the session, your Words Per Minute (WPM) and accuracy percentage will be displayed.

## Main Functions

- **Random Word Display**: The game displays random words for you to type.
- **Typing Input**: You type the displayed words into an input field.
- **Speed Measurement**: The game calculates your typing speed in Words Per Minute (WPM).
- **Accuracy Measurement**: The game calculates your typing accuracy as a percentage.
- **Results Display**: At the end of the session, your WPM and accuracy are displayed.

## How to Use

1. **Run the Application**: Open a terminal or command prompt and navigate to the directory where `main.py` is located. Run the following command to start the application:

   ```bash
   python main.py
   ```

2. **Start the Game**: Once the application window opens, click the "Start" button to begin the game.

3. **Typing Practice**: Type the word displayed on the screen into the input field and press "Enter" to submit. A new word will appear after each submission.

4. **Game Duration**: The game runs for 60 seconds. Try to type as many words as you can within this time.

5. **View Results**: After 60 seconds, the game will automatically end, and your WPM and accuracy percentage will be displayed on the screen.

## Troubleshooting

- **No Display Found Error**: If you encounter an error stating "No display found," ensure you are running the application in an environment with a graphical display (e.g., not a headless server).

- **Python Not Found**: If you receive an error that Python is not recognized, ensure Python is installed and added to your system's PATH.
