# Trivia Quiz Program

Welcome to the Trivia Quiz Program! This application is designed to provide an engaging and educational experience by presenting multiple-choice or short-answer questions on various topics. Users can enter their answers, and the game will track their score. At the end of the quiz, the total number of correct answers will be displayed.

## Main Functions

The Trivia Quiz Program consists of the following main components:

1. **Question Class**: Represents a single quiz question, including the question text, possible answers, and the correct answer. It also includes a method to check if a user's answer is correct.

2. **Quiz Class**: Manages the quiz process, including loading questions, presenting them to the user, tracking the score, and displaying the final results.

3. **Main Module**: Initializes the quiz with a set of questions and starts the quiz process.

## Installation

To run the Trivia Quiz Program, you need to have Python installed on your system. Follow these steps to set up the environment and run the application:

### Step 1: Install Python

If you haven't already, download and install Python from the official website: [python.org](https://www.python.org/downloads/). Follow the instructions for your operating system.

### Step 2: Set Up the Environment

1. **Clone the Repository**: Download or clone the repository containing the Trivia Quiz Program files to your local machine.

2. **Navigate to the Project Directory**: Open a terminal or command prompt and navigate to the directory where the program files are located.

### Step 3: Run the Application

1. **Execute the Main Module**: In the terminal or command prompt, run the following command to start the quiz:

   ```bash
   python main.py
   ```

2. **Follow the On-Screen Instructions**: The program will present questions one by one. Enter the number corresponding to your chosen answer and press Enter.

3. **View Your Score**: After answering all questions, the program will display your total score.

## How to Play

1. **Start the Quiz**: Run the `main.py` file to begin the quiz.

2. **Answer Questions**: For each question, you will see a list of options. Enter the number of your chosen answer and press Enter.

3. **Receive Feedback**: After each question, the program will inform you whether your answer was correct or incorrect.

4. **Complete the Quiz**: Once all questions have been answered, the program will display your total score.
