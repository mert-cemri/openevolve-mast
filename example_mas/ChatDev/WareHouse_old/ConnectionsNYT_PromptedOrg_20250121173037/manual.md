# Puzzle Game User Manual

Welcome to the Puzzle Game! This manual will guide you through the installation process, introduce the main functions of the software, and provide instructions on how to play the game.

## Introduction

The Puzzle Game is an interactive application where users are challenged to group sixteen words into four categories of four words each. The game follows the standard logic of the NYT Connections game. After each guess, the game will indicate if the chosen words form a valid group. The game ends when the user forms all four correct groups or runs out of tries.

## Main Functions

- **Game Initialization**: The game starts by initializing the puzzle and setting up the user interface.
- **User Interaction**: The game interacts with the user through a command-line interface, prompting for guesses and providing feedback.
- **Guess Validation**: The game checks each guess to determine if it forms a valid group of words.
- **Win/Loss Conditions**: The game ends when all categories are correctly guessed or the user runs out of tries.

## Installation

### Prerequisites

- Python 3.x installed on your system.

### Installation Steps

1. **Clone the Repository**: Download or clone the repository containing the game files to your local machine.

2. **Navigate to the Project Directory**: Open a terminal and navigate to the directory where the game files are located.

3. **Install Dependencies**: The game does not require any external dependencies, so you can skip this step. However, ensure that your Python environment is set up correctly.

## How to Play

1. **Start the Game**: Run the `main.py` file to start the game. You can do this by executing the following command in your terminal:

   ```bash
   python main.py
   ```

2. **Game Instructions**: Upon starting, the game will display a welcome message and instructions on how to play.

3. **Enter Your Guess**: You will be prompted to enter four words separated by spaces. Make sure to enter exactly four words.

4. **Receive Feedback**: After each guess, the game will inform you whether your guess was correct or incorrect.

5. **Continue Guessing**: Keep guessing until you either solve all categories or exhaust your tries.

6. **End of Game**: The game will display a message indicating whether you have won or lost.

## Conclusion

Enjoy playing the Puzzle Game! Challenge yourself to group the words correctly and improve your logical thinking skills. If you encounter any issues or have questions, feel free to reach out for support. Happy puzzling!
