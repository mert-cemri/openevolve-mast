# Trivia Quiz Program

Welcome to the Trivia Quiz Program! This application is designed to provide an engaging and interactive trivia quiz experience, where users can answer multiple-choice or short-answer questions on various topics. The game tracks the user's score and displays the total correct answers at the end of the quiz.

## Main Functions

- **Interactive Quiz Interface**: The program presents questions to the user and allows them to submit answers through a graphical user interface (GUI) built with Tkinter.
- **Multiple-Choice and Short-Answer Questions**: The quiz supports both multiple-choice questions, where users select an answer from a list of options, and short-answer questions, where users type their answer.
- **Score Tracking**: The program keeps track of the user's score, updating it with each correct answer.
- **Quiz Completion Summary**: At the end of the quiz, the program displays the user's total score and the number of questions answered correctly.

## Installation

### Environment Setup

The Trivia Quiz Program is developed in Python and utilizes the Tkinter library, which is part of the Python standard library. Therefore, no additional external dependencies are required.

### Steps to Install

1. **Ensure Python is Installed**: Make sure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

2. **Clone the Repository**: Download or clone the project repository to your local machine.

3. **Navigate to the Project Directory**: Open a terminal or command prompt and navigate to the directory where the project files are located.

4. **Run the Program**: Execute the following command to start the quiz application:
   ```bash
   python main.py
   ```

## How to Use/Play

1. **Launch the Application**: Run the `main.py` file to start the Trivia Quiz Program. A window will open displaying the quiz interface.

2. **Answer Questions**: 
   - For multiple-choice questions, click on the button corresponding to your chosen answer.
   - For short-answer questions, type your answer into the text entry field and click the "Submit" button.

3. **Receive Feedback**: After submitting an answer, the program will indicate whether your answer was correct or incorrect. If incorrect, the correct answer will be displayed.

4. **Proceed to Next Question**: Click the "Next" button to move on to the next question.

5. **Complete the Quiz**: Continue answering questions until the quiz is complete. At the end, your total score and the number of correct answers will be displayed.

6. **End of Quiz**: Once the quiz is over, the application will display your final score and thank you for playing.

Enjoy testing your knowledge with the Trivia Quiz Program!
