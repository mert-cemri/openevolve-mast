## Introduction

The Pong Game is a classic arcade-style game where two players control paddles to bounce a ball back and forth. The objective is to score points by making the opponent miss the ball. This game is developed using Python and the Pygame library, providing a simple yet engaging experience.

## Installation

To run the Pong Game, you need to have Python and Pygame installed on your system. Follow the steps below to set up your environment:

### Step 1: Install Python

Ensure you have Python installed on your system. You can download and install Python from the official website: [Python.org](https://www.python.org/downloads/).

### Step 2: Install Pygame

Once Python is installed, you can install Pygame using pip, the Python package manager. Open your terminal or command prompt and run the following command:

```bash
pip install pygame
```

## Game Setup

After installing the necessary dependencies, you can set up the game by following these steps:

1. Download the game files (`main.py`, `game.py`, `paddle.py`, `ball.py`) and place them in the same directory.

2. Open a terminal or command prompt and navigate to the directory containing the game files.

3. Run the game by executing the following command:

```bash
python main.py
```

The game window should open, and you can start playing!

## How to Play

The Pong Game is designed for two players. Each player controls a paddle to bounce the ball back and forth. Here are the controls:

- **Player 1 Controls:**
  - Move Up: Press the `W` key
  - Move Down: Press the `S` key

- **Player 2 Controls:**
  - Move Up: Press the `Up Arrow` key
  - Move Down: Press the `Down Arrow` key

The objective is to score points by making the opponent miss the ball. The game continues until you decide to quit.

## Game Features

- **Two-Player Mode:** Enjoy the classic Pong experience with a friend.
- **Simple Controls:** Easy-to-use controls for both players.
- **Score Tracking:** The game keeps track of each player's score.
- **Realistic Ball Movement:** The ball bounces off paddles and edges, simulating realistic physics.
