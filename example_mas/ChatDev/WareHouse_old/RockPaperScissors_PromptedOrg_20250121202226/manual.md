# Rock-Paper-Scissors Game

Welcome to the Rock-Paper-Scissors game! This is a simple, interactive game where you can play against the computer. The game follows the traditional rules of Rock-Paper-Scissors: Rock beats Scissors, Scissors beats Paper, and Paper beats Rock. You can play as many rounds as you like!

## Quick Install

To play the Rock-Paper-Scissors game, you need to have Python installed on your computer. If you don't have Python installed, you can download it from [python.org](https://www.python.org/).

Once Python is installed, you can run the game script directly as it does not require any additional dependencies.

## How to Play

1. **Download the Game Script:**
   - Save the provided `main.py` script to your computer.

2. **Run the Game:**
   - Open a terminal or command prompt.
   - Navigate to the directory where you saved `main.py`.
   - Run the script by typing `python main.py` and pressing Enter.

3. **Game Instructions:**
   - When the game starts, you will be prompted to enter your choice: Rock, Paper, or Scissors.
   - Type your choice and press Enter.
   - The computer will randomly select its choice.
   - The game will display both your choice and the computer's choice, and announce the winner of the round.
   - You will be asked if you want to play again. Type 'yes' to play another round or 'no' to exit the game.

4. **End the Game:**
   - When you decide not to play anymore, type 'no' when asked if you want to play again. The game will thank you for playing and exit.
