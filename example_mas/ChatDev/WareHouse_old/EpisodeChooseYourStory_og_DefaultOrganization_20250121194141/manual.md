# Interactive Storytelling Game

Welcome to the Interactive Storytelling Game! This application allows players to immerse themselves in a narrative where their choices influence the story's progression and outcome. The game tracks variables such as relationships and items, providing different paths and endings based on player decisions.

## Main Functions

- **Narrative Segments**: Experience a rich storyline presented in segments.
- **Branching Choices**: Make decisions that affect the story's direction.
- **Dynamic Outcomes**: Player choices influence relationships and items, altering possible endings.
- **Graphical User Interface**: Engage with the game through an intuitive GUI built with Tkinter.

## Installation Guide

### Prerequisites

- **Python**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Installation Steps

1. **Clone the Repository**: Download the game code from the repository.
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Install Dependencies**: The game uses Tkinter, which is part of the Python standard library, so no additional dependencies are required. However, ensure your Python environment is set up correctly.

3. **Run the Game**: Execute the main module to start the game.
   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Launch the game by running the `main.py` script. The main window will appear, displaying the initial narrative segment.

2. **Make Choices**: Read the narrative and choose from the available options. Click on the buttons corresponding to your choice to progress the story.

3. **Track Progress**: Your decisions will influence relationships and items, which are tracked throughout the game. These variables will affect the story's outcome.

4. **Explore Different Paths**: Replay the game to explore different choices and discover various endings.

## Troubleshooting

- **Unknown Destination Error**: If you encounter an error message stating "The chosen path leads to an unknown destination," it means the selected choice does not lead to a valid scene. Please choose another option.

## Additional Information

For further assistance or to report issues, please contact our support team or visit our [GitHub repository](<repository-url>) for more information.

Enjoy your adventure in the Interactive Storytelling Game!
