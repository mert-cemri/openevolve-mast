# Close Elements Checker User Manual

## Introduction
The Close Elements Checker is a simple GUI application that checks if there are any two numbers in a given list that are closer to each other than a given threshold. This manual provides instructions on how to install the necessary dependencies and how to use the application.

## Installation
To use the Close Elements Checker, you need to install the following dependencies:

- Python 3.x
- tkinter library

You can install the dependencies using pip or conda. Here are the installation commands:

### Pip
```
pip install tkinter
```

### Conda
```
conda install -c conda-forge tkinter
```

## Usage
1. Open the terminal or command prompt.
2. Navigate to the directory where the Close Elements Checker files are located.
3. Run the following command to start the application:
```
python main.py
```
4. The Close Elements Checker window will open.
5. Enter the numbers separated by spaces in the "Numbers" field.
6. Enter the threshold value in the "Threshold" field.
7. Click the "Check" button to see the result.
8. A message box will appear with the result indicating whether there are close elements or not.

## Example
### Input
Numbers: 1.0 2.8 3.0 4.0 5.0 2.0
Threshold: 0.3

### Output
There are close elements: True

## Conclusion
The Close Elements Checker is a simple and easy-to-use application that allows you to check if there are any two numbers in a given list that are closer to each other than a specified threshold. By following the instructions in this manual, you can successfully install and use the Close Elements Checker application.
