# Close Elements Checker
This is a simple GUI application that checks if there are any two numbers in a given list that are closer to each other than a given threshold.
## Usage
1. Enter the numbers separated by spaces in the "Numbers" field.
2. Enter the threshold value in the "Threshold" field.
3. Click the "Check" button to see the result.
## Example
### Input
Numbers: 1.0 2.8 3.0 4.0 5.0 2.0
Threshold: 0.3
### Output
There are close elements: True
