# Word Grouping Puzzle Game

Welcome to the Word Grouping Puzzle Game! This application challenges you to group sixteen words into four categories of four words each, following the standard logic of the NYT Connections game. After each guess, you'll receive feedback on whether the chosen words form a valid group. The game concludes when you form all four correct groups or run out of attempts.

## Main Functions

- **Word Grouping Puzzle**: The core functionality of the game is to allow users to select words and group them into categories. The game provides feedback on the correctness of the groupings.
- **Graphical User Interface (GUI)**: The game features a user-friendly interface built with Tkinter, allowing easy interaction with the game elements.
- **Feedback System**: After each guess, the game informs you if the selected words form a correct group and provides hints for incorrect guesses.

## Installation

### Environment Setup

This application is built using Python and requires no external dependencies. Ensure you have Python installed on your system. You can download Python from [python.org](https://www.python.org/downloads/).

### Running the Game

1. **Clone the Repository**: Download or clone the repository containing the game code to your local machine.

2. **Navigate to the Project Directory**: Open a terminal or command prompt and navigate to the directory where the game files are located.

3. **Run the Game**: Execute the following command to start the game:

   ```bash
   python main.py
   ```

## How to Play

1. **Start the Game**: Launch the game by running the `main.py` file. A window will open displaying the words you need to group.

2. **Select Words**: Click on the words to select them. You must select exactly four words to form a group.

3. **Submit Your Guess**: Once you have selected four words, click the "Submit" button to check if your selection forms a valid group.

4. **Receive Feedback**: The game will inform you if your guess is correct. If incorrect, it will highlight the words that belong to the correct categories.

5. **Game Over**: The game ends when you successfully form all four groups or exhaust your attempts. A message will display your success or indicate that you've run out of attempts.

6. **Restart**: To play again, simply restart the application.

## Troubleshooting

- **Invalid Guess**: Ensure you select exactly four words before submitting your guess.
- **Game Not Starting**: Verify that Python is correctly installed and that you are running the command in the correct directory.

Enjoy the challenge and have fun grouping words into their respective categories!
