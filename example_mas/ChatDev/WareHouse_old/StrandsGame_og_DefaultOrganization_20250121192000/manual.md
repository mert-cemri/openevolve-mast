# Strands Puzzle

Welcome to the Strands Puzzle, a word-segmentation game where players combine strands of text to form meaningful words or phrases. This guide will help you understand the main functions of the software, how to set up the environment, and how to play the game.

## Main Functions

The Strands Puzzle application provides the following main functions:

- **Interactive Gameplay**: Players are presented with multiple strands of text. The goal is to combine these strands to form valid words or phrases.
- **Solution Verification**: The program checks the player's input against the correct solution and provides feedback on whether the puzzle is solved correctly.
- **User-Friendly Interface**: The application features a graphical user interface (GUI) built using Tkinter, making it easy to interact with the game.

## Installation

### Environment Setup

The Strands Puzzle is developed in Python and requires a Python environment to run. Follow these steps to set up your environment:

1. **Install Python**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

2. **Clone the Repository**: Download the source code for the Strands Puzzle.

3. **Navigate to the Project Directory**: Open a terminal or command prompt and navigate to the directory where the source code is located.

4. **Install Dependencies**: The project does not require any external dependencies, so you can proceed to run the application without additional installations.

## How to Play

1. **Launch the Application**: Run the `main.py` file to start the Strands Puzzle. You can do this by executing the following command in your terminal or command prompt:
   ```bash
   python main.py
   ```

2. **Understand the Objective**: The main objective is to combine the given strands of text to form meaningful words or phrases.

3. **Select Strands**: Click on the buttons representing different strands to select them. The selected strands will appear in the "Selected" section.

4. **Submit Your Solution**: Once you believe you have the correct combination, click the "Submit" button to check your solution.

5. **Receive Feedback**: The application will inform you if your solution is correct or incorrect. If incorrect, you can try again by selecting different strands.

6. **Clear Selection**: After each submission, the selected strands will be cleared, allowing you to start fresh for the next attempt.

## Troubleshooting

- **GUI Initialization Error**: If you encounter an error related to GUI initialization, ensure you are running the application in a graphical environment that supports Tkinter.

- **Python Version Compatibility**: Ensure you are using a compatible version of Python. The application is designed to work with Python 3.x.

Enjoy playing the Strands Puzzle and challenge yourself to solve the word-segmentation puzzles!
