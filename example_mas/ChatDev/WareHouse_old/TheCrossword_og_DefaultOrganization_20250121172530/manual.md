# Crossword Puzzle Application

Welcome to the Crossword Puzzle Application! This application allows you to engage in a fun and interactive crossword puzzle experience. You can fill in the crossword grid by entering words based on the provided clues for both across and down entries. The application will validate your entries and notify you upon successful completion of the puzzle.

## Main Functions

- **Crossword Grid**: A grid of squares where you can input letters to form words.
- **Clues**: Each word in the grid is associated with a clue, which helps you determine the correct word to fill in.
- **Validation**: The application checks if the letters you enter match the correct words.
- **Completion Notification**: Once all words are correctly filled in, the application congratulates you on completing the puzzle.

## Installation

### Environment Setup

This application is built using Python and requires no external dependencies, making it simple to set up and run. Ensure you have Python installed on your system.

### Steps to Install

1. **Clone the Repository**: Download the application code from the repository.
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Run the Application**: Execute the main application file using Python.
   ```bash
   python main.py
   ```

   Ensure you are running this on a system with a graphical display, as the application uses a GUI.

## How to Use

1. **Launch the Application**: Run the `main.py` file to start the crossword puzzle application. A window will open displaying the crossword grid.

2. **Understanding the Interface**:
   - The grid consists of empty squares where you can input letters.
   - Clues are provided for each word, indicating the direction (across or down) and the starting position in the grid.

3. **Entering Words**:
   - Click on a square in the grid to start typing.
   - Enter letters corresponding to the word for the given clue.
   - The application automatically validates each letter as you type.

4. **Submitting Words**:
   - Once you have filled in the grid, click the "Submit" button.
   - The application will check your entries against the correct answers.

5. **Completion**:
   - If all words are correct, a congratulatory message will appear.
   - If any words are incorrect, the application will notify you to try again.

Enjoy solving the crossword puzzle and challenge yourself with different clues and words!
