# Responsive Web-Based Calculator

Welcome to the user manual for the Responsive Web-Based Calculator. This document will guide you through the main functions of the calculator, how to set up the environment, and how to use the calculator effectively.

## Main Functions

The Responsive Web-Based Calculator is a simple yet powerful tool designed to perform basic arithmetic operations. Here are its main features:

- **Numeric Keypad**: Allows you to input numbers from 0 to 9.
- **Basic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/).
- **Decimal Point**: Allows you to perform calculations with decimal numbers.
- **Display Area**: Shows the current input and results of calculations.
- **Responsive Design**: Ensures the calculator works seamlessly on various screen sizes, from mobile devices to desktop computers.

## Installation

The calculator is a web-based application, which means you don't need to install any software on your computer. All you need is a modern web browser such as Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge.

### Environment Setup

1. **Download the Files**: Ensure you have the `index.html` file containing the HTML structure, CSS styling, and JavaScript logic for the calculator.
2. **Open in Browser**: Simply open the `index.html` file in your preferred web browser.

## How to Use the Calculator

1. **Launch the Calculator**: Open the `index.html` file in your web browser to launch the calculator interface.
2. **Input Numbers**: Click on the numeric buttons (0-9) to input numbers. You can also use the decimal point (.) for decimal numbers.
3. **Perform Operations**: Click on the operation buttons (+, -, *, /) to select the desired arithmetic operation.
4. **Calculate**: After entering the numbers and selecting an operation, click the equals (=) button to perform the calculation and display the result.
5. **Clear Input**: If you make a mistake or want to start a new calculation, simply begin entering new numbers, and the display will update accordingly.

## Troubleshooting

- **Display Shows "Error"**: This occurs when you attempt to divide by zero. Clear the input and try a different calculation.
- **Responsive Issues**: Ensure your browser window is not too small. The calculator is designed to be responsive, but extremely small windows may affect its usability.
