# Match-3 Puzzle Game

Welcome to the Match-3 Puzzle Game, a fun and engaging game reminiscent of popular games like Candy Crush. This game allows you to swap adjacent candies to form matches of three or more, clear them from the board, and earn points. The board updates dynamically with new candies falling into place, providing endless entertainment.

## Main Functions

- **Swap Candies**: Click on two adjacent candies to swap them. If the swap results in a match of three or more candies of the same color, they will be cleared from the board.
- **Match and Clear**: Form matches of three or more candies to clear them from the board and earn points.
- **Dynamic Board Update**: After candies are cleared, new candies will fall from the top to fill the empty spaces, creating opportunities for new matches.
- **Score Tracking**: Your score is tracked and displayed on the screen, allowing you to challenge yourself to achieve higher scores.

## Installation

To run the Match-3 Puzzle Game, you need to have Python and the Pygame library installed on your system. Follow the steps below to set up your environment:

### Step 1: Install Python

Ensure that Python is installed on your system. You can download it from the [official Python website](https://www.python.org/downloads/).

### Step 2: Install Pygame

Pygame is a set of Python modules designed for writing video games. It provides functionalities like creating windows, drawing shapes, and handling user input.

To install Pygame, open your terminal or command prompt and run the following command:

```bash
pip install pygame>=2.0.0
```

Alternatively, you can use the `requirements.txt` file provided with the game:

```bash
pip install -r requirements.txt
```

## How to Play

1. **Start the Game**: Run the `main.py` file to start the game. You can do this by navigating to the directory containing the game files and executing the following command in your terminal or command prompt:

   ```bash
   python main.py
   ```

2. **Game Interface**: The game window will open, displaying an 8x8 grid filled with colorful candies.

3. **Swap Candies**: Use your mouse to click on two adjacent candies to swap them. The goal is to form a horizontal or vertical line of three or more candies of the same color.

4. **Score Points**: Each successful match will clear the candies from the board and add points to your score. The current score is displayed at the top of the game window.

5. **Continue Playing**: The game continues as new candies fall into place, allowing for continuous play and score accumulation.

6. **Exit the Game**: To exit the game, simply close the game window or press the 'X' button on the top right corner.

Enjoy playing the Match-3 Puzzle Game and challenge yourself to achieve the highest score possible!
