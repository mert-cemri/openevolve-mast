# Typing Practice Game

Welcome to the Typing Practice Game! This application is designed to help you improve your typing speed and accuracy by providing random phrases for you to type within a set time. At the end of each session, the game calculates your words-per-minute (WPM) and accuracy percentage, displaying the final results.

## Main Functions

- **Random Phrase Generation**: The game provides random phrases for you to type, ensuring a varied and engaging practice experience.
- **Timing**: The game includes a timer to measure how long it takes you to type the given phrase.
- **Performance Metrics**: After you complete typing, the game calculates and displays your raw WPM, adjusted WPM (considering accuracy), and accuracy percentage.

## Installation

To run the Typing Practice Game, you need to have Python installed on your system. Follow these steps to set up the environment:

1. **Install Python**: If you haven't already, download and install Python from [python.org](https://www.python.org/downloads/).

2. **Clone the Repository**: Clone the repository containing the game code to your local machine.

   ```bash
   git clone <repository-url>
   ```

3. **Navigate to the Project Directory**: Change your working directory to the project folder.

   ```bash
   cd <project-directory>
   ```

4. **Install Dependencies**: The game does not require any external Python packages, so no additional dependencies need to be installed.

## How to Play

1. **Start the Game**: Run the main Python file to start the game.

   ```bash
   python main.py
   ```

2. **Type the Displayed Phrase**: A random phrase will be displayed on the screen. Type the phrase as accurately and quickly as possible.

3. **Submit Your Input**: Press `Enter` after typing the phrase to submit your input.

4. **View Your Results**: The game will calculate and display your performance metrics, including:
   - **Raw Words per Minute (WPM)**: The number of words you typed per minute.
   - **Adjusted Words per Minute**: The raw WPM adjusted for accuracy.
   - **Accuracy Percentage**: The percentage of characters you typed correctly.

5. **Repeat**: Feel free to play multiple rounds to improve your typing skills!

## Additional Information

- The game is designed to be simple and user-friendly, making it accessible for typists of all skill levels.
- The phrases used in the game are stored in the `word_provider.py` file and can be customized if desired.

Enjoy improving your typing skills with the Typing Practice Game!
