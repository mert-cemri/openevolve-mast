## Main Functions

The UC Berkeley Website Prototype includes the following main components:

1. **Header**: Displays the university's brand elements and navigation links to major sections such as Admissions, Academics, Research, and News.

2. **Main Content**: Contains detailed sections for Admissions, Academics, and Research, along with a dedicated News & Events section.

3. **Footer**: Provides contact details and links to UC Berkeley's social media pages.

4. **Responsive Design**: Ensures the website is accessible and mobile-friendly, adapting to different screen sizes.

## Installation

To set up the environment and run the website prototype, follow these steps:

### Prerequisites

- Ensure you have Node.js and npm installed on your machine. You can download them from [Node.js official website](https://nodejs.org/).

### Steps

1. **Clone the Repository**

   Clone the project repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**

   Change into the project directory:

   ```bash
   cd <project-directory>
   ```

3. **Install Dependencies**

   Install the necessary dependencies using npm:

   ```bash
   npm install
   ```

4. **Run the Application**

   Start the development server to view the website prototype:

   ```bash
   npm start
   ```

   This will open the application in your default web browser at `http://localhost:3000`.

## Usage

Once the application is running, you can interact with the website prototype:

- **Navigate** through the main sections using the header links.
- **Explore** the content in each section, including Admissions, Academics, Research, and News & Events.
- **Contact** UC Berkeley or follow their social media pages using the links in the footer.

## Accessibility and Mobile-Friendly Design

The website prototype is designed with accessibility in mind, using semantic HTML elements and ARIA labels to enhance navigation for users with disabilities. The responsive design ensures optimal viewing on various devices, including desktops, tablets, and smartphones.
