# Number Guessing Game

Welcome to the Number Guessing Game! This is a simple command-line interface (CLI) game where you try to guess a randomly generated number. The game provides hints to guide you until you guess the correct number.

## Main Functions

- **Random Number Generation**: The game generates a random number between 1 and 100.
- **User Interaction**: You will be prompted to guess the number, and the game will provide feedback whether your guess is too high, too low, or correct.
- **Hints**: After each guess, the game will tell you if the correct number is higher or lower than your guess.
- **Winning Condition**: The game ends when you correctly guess the number.

## Installation

This game is implemented in Python and does not require any external dependencies. Ensure you have Python installed on your system.

### Step-by-step Installation Guide

1. **Install Python**: If you haven't already, download and install Python from [python.org](https://www.python.org/downloads/).

2. **Clone the Repository**: Download the game files to your local machine. You can do this by cloning the repository or downloading the files directly.

3. **Navigate to the Game Directory**: Open your terminal or command prompt and navigate to the directory where the game files are located.

## How to Play

1. **Start the Game**: Run the game by executing the following command in your terminal:

   ```bash
   python main.py
   ```

2. **Guess the Number**: You will be prompted to enter your guess. Type a number between 1 and 100 and press Enter.

3. **Receive Feedback**: After each guess, the game will tell you if the correct number is higher or lower than your guess.

4. **Win the Game**: Continue guessing until you find the correct number. The game will congratulate you once you guess correctly.

5. **Play Again**: To play again, simply run the `main.py` file once more.

## Troubleshooting

- **Invalid Input**: If you enter a non-numeric value, the game will prompt you to enter a valid number.
- **Python Errors**: Ensure that Python is correctly installed and that you are running the game in the correct directory.

Enjoy playing the Number Guessing Game and challenge yourself to guess the number in the fewest attempts possible!
