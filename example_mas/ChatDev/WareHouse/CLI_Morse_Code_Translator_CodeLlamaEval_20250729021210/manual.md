Welcome to the CLI Morse Code Translator!

This manual will guide you through the installation and usage of the CLI Morse Code Translator.

### Installation

To install the CLI Morse Code Translator, you will need to have Python 3.x installed on your system. You can download Python from the official Python website: <https://www.python.org/downloads/>.

Once you have Python installed, you can install the CLI Morse Code Translator by running the following command in your terminal or command prompt:
```
pip install cli-morse-code-translator
```
### Usage

Once the CLI Morse Code Translator is installed, you can use it by running the following command in your terminal or command prompt:
```
morse-code-translator
```
This will start the CLI Morse Code Translator and you will be prompted to enter a message to translate.

You can also use the `--help` option to see a list of available options and their descriptions:
```
morse-code-translator --help
```
### Translating Text to Morse Code

To translate a message from English to Morse code, simply enter the message you want to translate and press enter. The CLI Morse Code Translator will then translate the message into Morse code and display it on the screen.

For example, if you enter the message "Hello, World!", the CLI Morse Code Translator will translate it into the following Morse code:
```
.... . .-.. .-.. --- .-- --- .-. .-.. -..
```
### Translating Morse Code to Text

To translate a message from Morse code to English, simply enter the Morse code you want to translate and press enter. The CLI Morse Code Translator will then translate the Morse code into English and display it on the screen.

For example, if you enter the Morse code ".... . .-.. .-.. --- .-- --- .-. .-.. -..", the CLI Morse Code Translator will translate it into the following message:
```
Hello, World!
```
### Exiting the CLI Morse Code Translator

To exit the CLI Morse Code Translator, simply press `Ctrl + C` on your keyboard. This will exit the program and return you to the command prompt.
