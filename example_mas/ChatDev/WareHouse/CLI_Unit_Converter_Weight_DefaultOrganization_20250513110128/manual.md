# CLI Unit Converter for Weights/Mass

This application is a Command Line Interface (CLI) tool designed to convert weights/mass between different units. It supports conversions between kilograms (kg), pounds (lb), grams (g), and ounces (oz).

## Quick Install

This application does not require any external dependencies, making it straightforward to set up and use. Simply ensure you have Python installed on your system.

## ðŸ¤” What is this?

The CLI Unit Converter is a simple yet powerful tool that allows users to convert a given weight/mass value from one unit to another. This can be particularly useful for tasks involving international shipping, cooking, or any scenario where weight/mass conversion is necessary.

### Main Functions

- **Convert Units**: Convert a numerical value from a source unit to a target unit.
- **Supported Units**: Kilograms (kg), Pounds (lb), Grams (g), Ounces (oz).

## ðŸ“– How to Use

### Running the Application

1. **Download the Script**: Save the `main.py` script to your local machine.

2. **Open a Terminal**: Navigate to the directory where `main.py` is located.

3. **Execute the Script**: Run the following command in your terminal:
   ```bash
   python main.py
   ```

### Using the Application

1. **Input Value**: When prompted, enter the numerical value you wish to convert.

2. **Input Source Unit**: Enter the unit of the value you are converting from. Supported units are `kg`, `lb`, `g`, and `oz`.

3. **Input Target Unit**: Enter the unit you wish to convert the value to. Supported units are `kg`, `lb`, `g`, and `oz`.

4. **View Result**: The application will display the converted value in the target unit.

### Example

- **Input**:
  ```
  Enter the value to convert: 1000
  Enter the source unit (kg, lb, g, oz): g
  Enter the target unit (kg, lb, g, oz): lb
  ```

- **Output**:
  ```
  Result: 2.2046 lb
  ```

## Troubleshooting

- **Invalid Input**: If you enter a non-numeric value, the application will prompt you to enter a valid numeric value.
- **Invalid Units**: If you enter a unit that is not supported, the application will prompt you to enter a valid unit from the list of supported units.
