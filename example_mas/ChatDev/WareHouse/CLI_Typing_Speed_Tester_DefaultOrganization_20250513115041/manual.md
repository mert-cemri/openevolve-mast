# Typing Speed Tester

A Command Line Interface (CLI) application to test and improve your typing speed and accuracy.

## Introduction

The Typing Speed Tester is a simple yet effective tool designed to help users measure their typing speed in words per minute (WPM) and accuracy percentage. The application presents a random paragraph, which the user must type as quickly and accurately as possible. The results are calculated and displayed at the end of the test.

## Main Functions

- **Random Paragraph Generation**: The application provides a random paragraph for each typing test, ensuring a unique experience every time.
- **Typing Speed Calculation**: Measures the user's typing speed in words per minute (WPM).
- **Accuracy Calculation**: Evaluates the accuracy of the typed text compared to the original paragraph.

## Installation

### Prerequisites

Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Quick Install

1. Clone the repository or download the source code files.
2. Navigate to the directory containing the files.

To install the necessary dependencies, run the following command:

```bash
pip install -r requirements.txt
```

*Note: If there is no `requirements.txt` file, it means there are no external dependencies required for this application.*

## How to Use

1. Open your terminal or command prompt.
2. Navigate to the directory where `main.py` is located.
3. Run the application using the following command:

```bash
python main.py
```

4. Follow the on-screen instructions:
   - The application will display a random paragraph.
   - Press "Enter" to start the test.
   - Type the paragraph as accurately and quickly as possible.
   - Press "Enter" again after you finish typing.

5. The application will display your results, including your typing speed in WPM and your accuracy percentage.
