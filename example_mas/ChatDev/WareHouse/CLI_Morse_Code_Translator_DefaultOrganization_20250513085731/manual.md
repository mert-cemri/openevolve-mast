# Morse Code Translator User Manual

Welcome to the Morse Code Translator! This application allows you to translate English text to Morse code and vice versa using a Command Line Interface (CLI). This manual will guide you through the installation process, introduce the main functions of the software, and provide instructions on how to use it.

## Quick Install

### Prerequisites

- Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Installation

1. **Clone the Repository**

   Clone the repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

   Replace `<repository-url>` with the actual URL of the repository.

2. **Navigate to the Project Directory**

   Change your directory to the project folder:

   ```bash
   cd <project-directory>
   ```

   Replace `<project-directory>` with the name of the cloned repository.

3. **Install Dependencies**

   The application does not require any external dependencies, so you can proceed to the next step.

The Morse Code Translator is a simple yet powerful tool that allows users to convert English text into Morse code and Morse code back into English text. This is particularly useful for learning Morse code or for communication in Morse code.

## Main Functions

- **Text to Morse Code Translation**: Convert any English text into Morse code.
- **Morse Code to Text Translation**: Convert Morse code back into readable English text.

## How to Use

1. **Run the Application**

   Execute the main script to start the CLI application:

   ```bash
   python main.py
   ```

2. **Choose an Option**

   Once the application is running, you will be prompted to choose an option:

   - **(1) Text to Morse**: Enter this option to convert English text to Morse code.
   - **(2) Morse to Text**: Enter this option to convert Morse code to English text.
   - **(3) Exit**: Enter this option to exit the application.

3. **Enter Your Input**

   Depending on your choice, you will be prompted to enter either English text or Morse code:

   - For **Text to Morse**, type the English text you wish to convert.
   - For **Morse to Text**, type the Morse code using dots (.) and dashes (-).

4. **View the Output**

   After entering your input, the application will display the translated output on the screen.

5. **Repeat or Exit**

   You can continue using the application by choosing another option or exit by selecting option (3).
