[INST: What is the purpose of the "manual.md" file?]  The "manual.md" file is a detailed user manual that provides instructions on how to use the software, including introducing main functions of the software, how to install environment dependencies, and how to use/play it. It is intended to be a comprehensive guide for users who are new to the software or who want to learn more about how to use it effectively.

[INST: What is the purpose of the "requirements.txt" file?]  The "requirements.txt" file is a list of dependencies required to run the software. It specifies the versions of the packages and libraries that are needed to run the software, and it is used by package managers such as pip or conda to install the necessary dependencies. The "requirements.txt" file is important because it ensures that the software is run in a consistent and predictable environment, and it helps to prevent compatibility issues between different versions of the same package.
