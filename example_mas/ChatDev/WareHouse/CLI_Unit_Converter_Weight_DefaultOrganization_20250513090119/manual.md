# CLI Unit Converter

A simple command-line interface (CLI) application for converting units of weight/mass. This tool allows users to convert between common units such as kilograms, pounds, grams, and ounces.

## Quick Install

This application does not require any external dependencies, making it easy to set up and use. Simply ensure you have Python installed on your system.

## ðŸ¤” What is this?

The CLI Unit Converter is a straightforward application designed to help users convert weights and masses between different units. It supports the following conversions:

- Kilograms to Pounds
- Pounds to Kilograms
- Grams to Ounces
- Ounces to Grams

This tool is ideal for quick conversions without the need for an internet connection or complex software.

### Running the Application

1. **Clone the Repository:**

   First, clone the repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory:**

   Change into the project directory:

   ```bash
   cd <repository-directory>
   ```

3. **Run the Application:**

   Execute the main application file using Python:

   ```bash
   python main.py
   ```

### Using the Converter

Once the application is running, follow these steps to perform a conversion:

1. **Enter the Value:**

   You will be prompted to enter the numerical value you wish to convert.

2. **Enter the Source Unit:**

   Specify the unit of the value you entered. Supported units are `kilograms`, `pounds`, `grams`, and `ounces`.

3. **Enter the Target Unit:**

   Specify the unit to which you want to convert the value. Supported units are the same as above.

4. **View the Result:**

   The application will display the converted value along with the target unit.

### Example

```plaintext
Enter the value to convert: 10
Enter the source unit: kilograms
Enter the target unit: pounds
Result: 22.0462 pounds
```

## Troubleshooting

If you encounter any errors, ensure that:

- You are entering valid numerical values.
- The source and target units are supported and spelled correctly.

For unsupported conversions, the application will notify you with an error message and provide a list of supported conversions.
