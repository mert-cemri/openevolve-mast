# Typing Speed Tester

A simple command-line interface (CLI) application to test your typing speed and accuracy.

## Introduction

The Typing Speed Tester is a Python-based application that presents a random paragraph for you to type. It calculates your typing speed in words per minute (WPM) and your typing accuracy percentage. This tool is perfect for anyone looking to improve their typing skills in a fun and interactive way.

## Quick Install

Since this project does not require any external dependencies, you can run it directly with Python. Ensure you have Python installed on your system.

## How to Run the Application

1. **Clone the Repository:**

   First, clone the repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

   Replace `<repository-url>` with the actual URL of the repository.

2. **Navigate to the Project Directory:**

   Change your directory to the project folder:

   ```bash
   cd <project-directory>
   ```

   Replace `<project-directory>` with the name of the cloned directory.

3. **Run the Application:**

   Execute the application using Python:

   ```bash
   python main.py
   ```

   This command will start the Typing Speed Tester.

## How to Use the Typing Speed Tester

1. **Start the Test:**

   Once you run the application, it will display a random paragraph for you to type. The timer starts as soon as the paragraph is displayed.

2. **Type the Paragraph:**

   Type the paragraph exactly as it appears. Press `Enter` when you finish typing.

3. **View Your Results:**

   After you submit your input, the application will calculate and display your typing speed in words per minute (WPM) and your typing accuracy as a percentage.

## Main Functions

- **Random Paragraph Generation:**

  The application randomly selects a paragraph from a predefined list for you to type.

- **Typing Speed Calculation:**

  It calculates your typing speed in words per minute (WPM) based on the time taken and the number of words typed.

- **Accuracy Calculation:**

  The application calculates your typing accuracy by comparing your input with the original paragraph.

## Conclusion

The Typing Speed Tester is a straightforward and effective tool to help you improve your typing skills. Whether you're a beginner or an experienced typist, this application provides a fun way to practice and track your progress. Enjoy typing!
