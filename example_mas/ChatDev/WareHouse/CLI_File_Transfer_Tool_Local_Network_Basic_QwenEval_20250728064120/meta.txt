 ### Task Breakdown

1. **Understand the Requirements:**
   - Create a basic CLI (Command Line Interface) file transfer tool.
   - The tool should work over a local network.
   - It should allow sending a file from one machine (client) to another (server).
   - The server should run a corresponding receiver script.

2. **Choose the Technology Stack:**
   - For simplicity, we can use Python, which has built-in libraries for networking and file handling.
   - We will use `socket` for network communication.

3. **Design the Architecture:**
   - **Server:** Listens for incoming connections, receives files, and saves them to a specified directory.
   - **Client:** Connects to the server, sends a file, and closes the connection.

4. **Implementation Steps:**
   - Write the server script to listen for incoming connections and receive files.
   - Write the client script to connect to the server and send files.

### Implementation

#### Server Script

The server script will:
- Bind to a specific IP and port.
- Listen for incoming connections.
- Accept a connection and receive a file.
- Save the received file to a specified directory.

```python
import socket
import os

def start_server(host='127.0.0.1', port=65432, save_directory='received_files'):
    os.makedirs(save_directory, exist_ok=True)
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.bind((host, port))
        s.listen()
        print(f"Server listening on {host}:{port}")
        conn, addr = s.accept()
        with conn:
            print(f"Connected by {addr}")
            file_name = conn.recv(1024).decode()
            file_path = os.path.join(save_directory, file_name)
            with open(file_path, 'wb') as f:
                while True:
                    data = conn.recv(1024)
                    if not data:
                        break
                    f.write(data)
            print(f"File {file_name} received and saved to {file_path}")

if __name__ == "__main__":
    start_server()
```

#### Client Script

The client script will:
- Connect to the server.
- Send the file name.
- Send the file content.
- Close the connection.

```python
import socket

def send_file(host='127.0.0.1', port=65432, file_path='example.txt'):
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.connect((host, port))
        file_name = os.path.basename(file_path)
        s.sendall(file_name.encode())
        with open(file_path, 'rb') as f:
            while True:
                data = f.read(1024)
                if not data:
                    break
                s.sendall(data)
        print(f"File {file_name} sent to {host}:{port}")

if __name__ == "__main__":
    send_file()
```

### Usage

1. **Start the Server:**
   - Run the server script on the machine that will receive the file.
   - Ensure the server is running before starting the client.

2. **Send a File:**
   - Run the client script on the machine that will send the file.
   - Specify the correct server IP and port, and the path to the file you want to send.

### Testing

- Ensure both scripts are running on different machines connected to the same local network.
- Verify that the file is sent and received correctly.
