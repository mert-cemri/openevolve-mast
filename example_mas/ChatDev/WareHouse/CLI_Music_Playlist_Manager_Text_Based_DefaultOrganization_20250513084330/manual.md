# Music Playlist Manager

Welcome to the Music Playlist Manager! This application is a command-line interface (CLI) tool that allows users to manage their music playlists efficiently. Users can create playlists, add songs, remove songs, and view their playlists. All data is stored in text files for simplicity and ease of access.

## Main Functions

1. **Create Playlist**: Users can create a new playlist by providing a name for it.
2. **Delete Playlist**: Users can delete an existing playlist by specifying its name.
3. **Add Song to Playlist**: Users can add a song to a specific playlist by providing the song's title, artist, and album.
4. **Remove Song from Playlist**: Users can remove a song from a specific playlist by specifying the song's title.
5. **View Playlists**: Users can view all their playlists and the songs contained within each playlist.

## Installation

### Requirements

This project does not require any external dependencies, making it easy to set up and use. It is built using Python, so ensure you have Python installed on your system.

### Quick Install

1. **Clone the Repository**: Clone the repository to your local machine using the following command:
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**: Change your directory to the project folder:
   ```bash
   cd <project-directory>
   ```

3. **Run the Application**: Execute the main script to start the CLI application:
   ```bash
   python main.py
   ```

## How to Use

1. **Launch the Application**: Run the `main.py` script to start the Music Playlist Manager.

2. **Main Menu**: You will be presented with a menu of options:
   - Enter `1` to create a new playlist.
   - Enter `2` to delete an existing playlist.
   - Enter `3` to add a song to a playlist.
   - Enter `4` to remove a song from a playlist.
   - Enter `5` to view all playlists.
   - Enter `6` to exit the application.

3. **Follow Prompts**: After selecting an option, follow the prompts to enter the necessary information (e.g., playlist name, song details).

4. **Exit**: Choose option `6` to exit the application when you are done managing your playlists.

## Data Storage

All playlists and songs are stored in a text file named `playlists.txt`. This file is automatically updated whenever you create, delete, add, or remove items from your playlists.
