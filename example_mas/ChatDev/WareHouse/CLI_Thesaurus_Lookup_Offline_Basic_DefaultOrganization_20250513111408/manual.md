# CLI Thesaurus Lookup Tool

Welcome to the CLI Thesaurus Lookup Tool, a command-line application designed to provide quick and easy access to synonyms and antonyms for any given word using a local, offline thesaurus file.

## Main Functions

- **Synonym Lookup**: Enter a word to receive a list of synonyms.
- **Antonym Lookup**: Enter a word to receive a list of antonyms.
- **User-Friendly Interface**: Simple command-line interface for ease of use.

## Installation

### Prerequisites

- **Python**: Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Quick Install

1. **Clone the Repository**: Download the source code to your local machine.
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Set Up Environment**: No external Python packages are required for this project. Ensure you have Python installed.

3. **Prepare Thesaurus Data**: Ensure the `thesaurus.json` file is in the same directory as the scripts. This file contains the thesaurus data in JSON format.

## How to Use

1. **Run the Application**: Execute the main script to start the CLI tool.
   ```bash
   python main.py
   ```

2. **Using the Tool**:
   - Upon starting, you will be greeted with a welcome message.
   - Enter a word to look up its synonyms and antonyms.
   - Type 'exit' to quit the application.

3. **Example Usage**:
   - Input: `Enter a word (or 'exit' to quit): happy`
   - Output:
     ```
     Synonyms for 'happy': joyful, content, pleased
     Antonyms for 'happy': sad, unhappy, miserable
     ```

## Additional Information

- **Customization**: You can modify the `thesaurus.json` file to add more words and their respective synonyms and antonyms.
- **Support**: For any issues or questions, please contact our support team.

Enjoy using the CLI Thesaurus Lookup Tool to enhance your vocabulary and writing!
